## Spider 1.0 · concert_singer

| Уровень | Ожидаемый вид | Промпт |
| --- | --- | --- |
| L1 | bar | Сравни количество певцов по странам. |
| L1 | line | Количество концертов по годам, лучше линейным графиком. |
| L1 | area | Покажи area chart: количество концертов по годам областным графиком. |
| L1 | bar | Покажи топ-5 стадионов по вместимости. |
| L2 | scatter | Построй scatter plot без агрегации: возьми из таблицы stadium колонку Capacity для X и колонку Average для Y; подпиши точки названием стадиона. |
| L3 | table | Для каждой страны певцов посчитай количество исполнителей, средний возраст, минимальный и максимальный возраст. Отсортируй страны по количеству певцов по убыванию и покажи результат таблицей. |
